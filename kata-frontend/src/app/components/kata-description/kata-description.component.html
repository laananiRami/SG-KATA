Kata FOO BAR QUIX

Description

Un algorithme qui transforme un nombre en chaîne de caractères selon les règles suivantes :

  • Si le nombre est divisible par 3 ou contient 3 → "FOO"
  • Si le nombre est divisible par 5 ou contient 5 → "BAR"
  • Si le nombre contient 7 → "QUIX"
  • Les règles "divisible par" sont prioritaires
